field

field: (__namedParameters: { datasource: DataSource; field: BlocksField }) => Promise<TinaBlocksField>
  • Defined in packages/gql/src/fields/blocks/index.ts:116

Resolves the values with their respective templates, specified by the template key.

// given
{
  name: 'sections',
  type: 'blocks',
  label: 'Sections',
  template_types: [ 'section' ]
}

// expect
{
  name: 'sections',
  type: 'blocks',
  label: 'Sections',
  template_types: [ 'section' ],
  component: 'blocks',
  templates: {
    sectionTemplateFields: {
      __typename: 'Section',
      label: 'Section',
      hide_body: false,
      fields: [Array]
    }
  },
  __typename: 'BlocksFormField'
}
