getTemplatesForSection

Returns the parsed templates for a given section. If no section is provided it returns a flattened array of all possible section templates

// Example
[
  {
    label: 'Post',
    hide_body: false,
    display_field: 'title',
    fields: [ {
      name: "title",
      label: "Title",
      type: "textarea",
      ...
    }]
    pages: [ 'posts/1.md' ]
  },
  ...
]
