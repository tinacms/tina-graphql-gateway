Cache

Cache: { build: <T>(gqlType: T) => T; builder: typeof builder; datasource: DataSource }
  • Defined in packages/gql/src/builder.ts:314

Holds an in-memory cache of GraphQL Objects which have been built, allowing re-use and avoiding name collisions

// ex. Any other uses of "SomeName" will return the cached version
cache.build(new GraphQLObjectType({name: 'SomeName', fields: {...}})
