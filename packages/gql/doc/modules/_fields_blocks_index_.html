field

  • field(__namedParameters: { cache: Cache; field: BlocksField }): Promise<GraphQLObjectType<BlocksField, any>>
    • Defined in packages/gql/src/fields/blocks/index.ts:62

    Builds a union type which adheres to the Tina Block shape.

    Since blocks need to be unique from one another depending on the templates they support, this is field builds a union which is namespaced by the template name:

    Given the following template definition:

    label: MyPage
fields:
- name: sections
  type: blocks
  label: Sections
  template_types:
    - cta
    - hero

    Builds:

    type MyPageSectionsBlocksField {
  name: String
  label: String
  component: String
  templates: SomeTemplateSectionsBlocksFieldTemplates
}
type SomeTemplateSectionsBlocksFieldTemplates {
  sectionTemplateFields: SectionForm
}
type SectionForm {
  fields: [MyPageSectionFormFields]
}
union MyPageSectionFormFields = CtaFormFields | HeroFormFields

    Resolves the values with their respective templates, specified by the template key.

    // given
{
  name: 'sections',
  type: 'blocks',
  label: 'Sections',
  template_types: [ 'section' ]
}

// expect
{
  name: 'sections',
  type: 'blocks',
  label: 'Sections',
  template_types: [ 'section' ],
  component: 'blocks',
  templates: {
    sectionTemplateFields: {
      __typename: 'Section',
      label: 'Section',
      hide_body: false,
      fields: [Array]
    }
  },
  __typename: 'BlocksFormField'
}
